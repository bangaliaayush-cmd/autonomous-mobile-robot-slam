# Autonomous Mobile Robot – SLAM (ROS 2)

## Overview

This project implements a simulated Autonomous Mobile Robot (AMR) using ROS 2 Humble. The robot is modeled as a differential-drive platform and simulated in Gazebo with LiDAR-based perception. Real-time mapping is performed using SLAM Toolbox to generate occupancy grid maps.

## Technologies Used

* ROS 2 Humble
* Gazebo Classic
* RViz
* SLAM Toolbox
* Python
* URDF / Xacro

## Features

* Differential-drive robot model
* Gazebo simulation
* LiDAR sensor integration
* Real-time SLAM mapping
* TF and odometry validation
* Occupancy grid map generation

## Results

Successfully generated and saved occupancy grid maps using LiDAR-based SLAM in a ROS 2 simulation environment.
